# Nazare website

Astro marketing site for Nazare.

## Commands

```sh
pnpm --filter @nazare/website dev
pnpm --filter @nazare/website build
pnpm --filter @nazare/website preview
```

## Docs route

`/docs` is reserved for documentation.

Future generated docs can target either:

- `src/content/docs` for content collections consumed by Astro pages.
- `src/pages/docs` for generated `.astro` or `.md` pages that map directly to routes.
